### [gh-dash](https://github.com/dlvhdr/gh-dash)

#### Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo:

```bash
git clone https://github.com/dracula/gh-dash.git
```

#### Install manually

Download using the [GitHub `.zip` download](https://github.com/dracula/gh-dash/archive/master.zip) option and unzip them.

#### Activating theme

1. cat ./gh-dash/config.yml >> ~/.config/gh-dash/config.yml
